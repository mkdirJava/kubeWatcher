# KubeWatcher

---

Tested on git bash

---

Prerequisite 

jq
kubectl 

---
To Use

place the kubeWatcher.sh on path

`
    source ./kubeWatcher.sh
    kubeWatcher <interval in seconds>
    eg
    kubeWatcher 5
`
The output gets printed to "~/.kubeWatcher.result1"
---
To run test

`

    ./test/_testRunner.sh

`
